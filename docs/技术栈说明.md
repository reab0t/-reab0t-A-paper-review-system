## 开发环境要求

### 前端开发
- Node.js >= 18.0.0
- npm >= 9.0.0
- VS Code 推荐插件：
  - Volar
  - ESLint
  - Prettier
  - TypeScript Vue Plugin

### 后端开发
- JDK 17 或更高版本
- Maven 3.8+
- IntelliJ IDEA 推荐插件：
  - Spring Boot Assistant
  - Lombok
  - SonarLint

### 数据库环境
- MySQL 8.0+
- Redis 6.2+
- Elasticsearch 7.x

### 容器环境
- Docker 24.0+
- Docker Compose 2.x
